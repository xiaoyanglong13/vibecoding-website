Setting up your workspace

Install Claude Code, get your accounts set up, and do your first agent run. The shortest module — by design.

What you'll be able to do

Have a working Claude Code installation on your laptop, with an account configured and your first "hello world" agent run completed.

Planned sub-pages

  • Install Claude Code. Mac and Windows instructions, account setup, API key configuration.
  • Your first agent run. A tiny task — "summarize the README in this folder" — to confirm the install works and you can read the output.
  • Appendix: OpenCode. The open-source alternative, when you might prefer it, and how it differs.
  • Which model to choose. Claude, GPT-4, Gemini, and beyond — a decision framework for comparing AI models across capability, cost, and speed, and how to pick the right one for a given task.
