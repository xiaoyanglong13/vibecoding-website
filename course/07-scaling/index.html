Module 7 · 90 minutes

Scaling up

You've built things solo. Here's how to build bigger — packaging your work, splitting tasks across multiple agents, and automating the repeatable parts.

What you'll be able to do

Package a workflow as a reusable skill, hand off sub-tasks to specialized subagents, set up hooks that run automatically, and coordinate small teams of agents working on different parts of the same problem.

Planned sub-pages

  • Skills. Turning a workflow you keep repeating into a packaged "skill" you (or a colleague) can invoke with one command.
  • Subagents. When and how to spin up specialized sub-agents — one researches, one drafts, one critiques.
  • Hooks. Automating the around-the-edges steps: format every report your agent writes, run tests every time it changes code.
  • Agent teams. Coordinating multiple agents on a larger problem. Where this works, where it doesn't (yet).
