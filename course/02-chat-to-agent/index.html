Module 2 · 45 minutes

From chat to agent

What changes when the AI can read your files, run code, and act in loops — and why that changes what's possible.

What you'll be able to do

Explain what an "agent" actually is, recognize when a task needs one versus a regular chatbot, and walk through one fully-worked end-to-end example before installing anything.

Planned sub-pages

  • The agent mental model. The "junior analyst with desktop access" analogy. What's new compared to ChatGPT: files, tools, loops.
  • A fully-worked example. Watch (via screenshots and transcripts) an agent take a vague request and produce a finished artifact, end to end.
  • When an agent helps and when it doesn't. A decision rubric: task shape, data location, repeatability.
