Syllabus

The course

Nine modules, roughly twelve hours of work end-to-end. Designed to be done in order, but built so you can drop in laterally via the reference and case sections.

Prerequisites

A laptop you can install software on. Comfort with spreadsheets. Willingness to spend 30 minutes on setup in Module 3. That's it.
