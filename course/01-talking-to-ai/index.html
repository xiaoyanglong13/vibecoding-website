Module 1 · 45 minutes

Talking to AI

Before we build anything, you need to be fluent in the basic conversation. This module covers what vibecoding actually is, how to write prompts that work, and ends with an exercise that produces a real artifact you can use.

What you'll be able to do

By the end of this module you'll be able to take a messy CSV file and turn it into a clean, summarized one — using only a chat interface and no code you wrote yourself. More importantly, you'll understand why the prompts you write succeed or fail, so you can apply the same patterns to your own problems.

How this module is structured

Three sub-pages, in order:

15 min · Concepts

1. What is vibecoding?

A working definition, what makes it different from regular ChatGPT use, and when it's the right tool.
20 min · Skills

2. Prompting basics

Four patterns that turn 80% of vague requests into useful outputs. Worked examples.
10 min · Practice

3. Exercises

Three hands-on exercises with sample data and answer keys. Produces a real artifact.

Why this comes first

Everything else in the course assumes you can write a clear, structured prompt. If you can't yet, the more powerful tools in later modules will amplify your confusion instead of your effectiveness. Spend the time here.
