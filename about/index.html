About this site

A practical course on vibecoding for business school students. Designed to be taught in class, taken independently online, or used as a reference.

Philosophy

Most "AI for business" content treats AI as a black box and teaches the prompts. This site treats AI as a tool you direct, and teaches the judgment — when to use it, how to know if it worked, when to stop. Business students don't need to become engineers, but they do need to be able to build, evaluate, and ship.

How to use this site

If you're taking the course from scratch, start at Module 0 and work straight through. If you're using it as a reference, the case library and reference section are organized for drop-in use. Search is across everything.

Contributing

If you've built something with vibecoding and would like to contribute it as a case, get in touch. The most useful cases come from real business problems people actually had — not invented examples.

Status

This site is in active development. Modules 0 and 1 are fully written; Modules 2–9 have outlines and will be filled in over time. The case library is growing case by case.
