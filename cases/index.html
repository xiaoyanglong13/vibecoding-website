Use case library

Business use cases

Real business problems with sample data, the prompts that solved them, and code you can copy and adapt. Browse by function.

Each case follows the same structure: the business decision being supported, the data available, the build, working code, what "good" looks like, and extensions. If you've worked through Modules 1–5, you have everything you need to follow along.

The one to start with

If you only have time to read one case, read Segmentation from a CSV. It's the simplest end-to-end example in the library and covers the core skills you'll use everywhere else.
